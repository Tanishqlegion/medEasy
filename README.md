# medEasy - Clinical AI Diagnostics Platform

**medEasy** is a next-generation healthcare platform that bridges hospitals and patients using advanced AI models. It transforms raw medical data into actionable clinical insights.

## 🚀 Project Status: Phase 1 (Landing Page)
This repository contains the **Minimum Viable Foundation** of the medEasy platform, focusing on the high-fidelity user interface and clinical aesthetic.

### 🏁 Features (Landing Page)
- **Neural DNA Background**: High-performance medical animation layer.
- **Dynamic Diagnostics Branding**: Modern, premium clinical UI components.
- **Universal Auth Flow**: Foundation for secure patient and hospital login.
- **Responsive Architecture**: Fully mobile-optimized clinical dashboard.

### 🛠️ Tech Stack
- **Frontend**: React + Vite
- **Styling**: Tailwind CSS 4.0 + Framer Motion (Animations)
- **Icons**: Lucide React
- **Backend (In-Progress)**: Node.js/Express + MongoDB + Python (ML)

## 📦 Getting Started
1. Clone the repository: `git clone https://github.com/Tanishqlegion/medEasy.git`
2. Install dependencies: `npm install`
3. Run developmental server: `npm run dev`
